# Scalable REST API with Authentication

## Features
- JWT Authentication
- Role-Based Access (User/Admin)
- CRUD Operations
- Secure API with Validation

## Tech Stack
Node.js, Express.js, MongoDB, React.js

## Setup
1. Clone repo
2. npm install
3. Add .env
4. npm start

## API Endpoints
- POST /api/auth/register
- POST /api/auth/login
- GET /api/tasks
...
